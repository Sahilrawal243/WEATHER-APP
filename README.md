# 🌦️ Weather App (CLI)

## Description

This is a simple Command Line Interface (CLI) Weather App built using Python.

The application allows users to enter a city name and displays the current weather information using the OpenWeatherMap API.

It is a beginner-friendly project that demonstrates how to use APIs, handle JSON data, and make HTTP requests in Python.
